Detailed Description

template<typename T>
struct ben::json::JSONSerializationInfo< T >

templated struct which contains information associated with the JSON serialization of a type

for all types T, the struct contains the following information:

  • JSONSerializationInfo<T>::SerializationFnType; an alias for a pointer to a function accepting a const T& as the single argument which returns a const std::u8string
  • JSONSerializationInfo<T>::serializable; a static constexpr bool which is true if type T is JSON serializable and false otherwise
  • JSONSerializationInfo<T>::serializer; a static constexpr function pointer of type JSONSerializationInfo<T>::SerializationFnType (depends on T) which points to the serialization implementation for type T if it exists or nullptr if it does not exist
  • JSONSerializationInfo<T>::serializer_impl(const T&); a static member function declaration for the implementation of the serialization of type T. If specialized via the helper macro, the JSONSerializationInfo<T>::serializer function pointer is set to the implementation method instead of nullptr
